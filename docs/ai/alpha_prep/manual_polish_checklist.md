Alpha polish manual checklist

- Startup screen: focus lands on the project name field when there are no recent projects.
- Startup screen: focus lands on the recent-projects list when recent projects are available.
- Startup screen: `Enter` in the project name field runs `CreateProjectCommand`.
- Rooms panel: `Enter` in the room name field runs `AddRoomCommand`.
- Editor shell: `Ctrl+N` creates a new project.
- Editor shell: `Ctrl+O` opens a project.
- Editor shell: `Ctrl+S` saves the current project.
- Editor shell: `Ctrl+Z` undoes the last action.
- Editor shell: `Ctrl+Y` redoes the last action.
- Editor shell: `Ctrl+Shift+Z` also redoes the last action.
- Editor shell: `Delete` removes the selected cabinet.
- Editor shell: `F1` opens the Alpha Limitations dialog.
